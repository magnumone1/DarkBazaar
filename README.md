  # 1.DarkBazaar
Rick and Morty themed mini e-commerce website built with HTML, CSS, and JavaScript.

## 2.About the Project
DarkBazaar is a mini online store inspired by the *Rick and Morty* universe.  
This project was created to practice frontend development, including layout design, JavaScript logic, and dynamic content rendering.

On the website, you can:
- Browse the product catalog
- Open product detail pages
- Navigate between pages without reloading
- View dynamically loaded content

##3.Technologies Used
- HTML5
- CSS3 (Flexbox / Grid)
- JavaScript (Vanilla JS)
- JSON (for dynamic product loading)
